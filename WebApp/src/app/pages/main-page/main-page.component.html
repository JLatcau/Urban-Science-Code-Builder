Help

You must provide this application with an input image. The input image should have the letters of the components structured in a 3 x 2 table format. Below is an example of what your image may look like.

This page provides you with two ways you can upload an image:

  • You are able to upload an image from your device by clicking the 'Upload From Device' button. This can be of file type png, jpg, and heic.
  • You can also take a picture of a peice of paper using the camera of device by clicking the 'Take Picture' button.
